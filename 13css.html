13. CSS Animations

I. @keyframes

Explanation: @keyframes se hum animation ka flow define karte hain — jaise element kaise start karega aur kaise end karega.

II. Animation Properties

Explanation: Animation ke liye kuch important properties hote hain:

  • animation-name: @keyframes ka naam
  • animation-duration: Animation kitni der chale
  • animation-delay: Kab start ho
  • animation-iteration-count: Kitni baar chale
  • animation-direction: normal, reverse, alternate
  • animation-fill-mode: Animation ke baad element ka state

III. Animation Timing Functions

Explanation: Timing functions define karti hain animation ki speed curve — jaise slow start, fast end, etc.

  • linear
  • ease
  • ease-in
  • ease-out
  • ease-in-out

IV. Infinite Loop Animations

Explanation: Jab aapko animation continuously chahiye ho, to animation-iteration-count: infinite; use karte hain.
