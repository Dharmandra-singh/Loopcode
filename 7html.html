7. HTML Images

I. Adding an Image

Explanation: Kisi image ko HTML page mein dikhane ke liye <img> tag ka use hota hai. Iska src attribute image ka URL batata hai ya fir tum image ka name extension ke sath likhar image ko same folder mai jisme code files hai usme rakhte ho to aise bhi image ko HTML page par dikhaya ja skta hai, aur alt attribute image na load hone par dikhne wala text hota hai ki yeh image kis chiz ki hai taki agr image load na ho to user aasani se samj sake ki yaha image kiski thi.

II. Image with Link

Explanation: Agar aap chahte ho ki image pe click karne se kisi dusre page par user chala jaye, toh image ko <a> tag ke andar rakhte hain.
