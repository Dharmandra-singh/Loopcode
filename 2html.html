2. Basic Structure of HTML

I. HTML Document Structure

Explanation: Jab hum koi HTML file banate hain, to ek fix structure follow karte hain. Ye structure browser ko batata hai ki kaunsa part kaise dikhana hai. Jaise ek ghar ka naksha hota hai, waise hi ek HTML document ka naksha hota hai.

II. DOCTYPE Declaration

Explanation: <!DOCTYPE html> batata hai ki document HTML5 ka hai. Agar hum ye nahi likhenge, to browser purane tareeke se page render karega ya kuch features kaam nahi karenge. Ye ek line sabse upar honi chahiye.

III. Head and Body Tags

Explanation:

  • Head: Is part me title, meta information (SEO, responsive design), CSS links, aur JavaScript ke links diye jaate hain. Jo cheeze user ko directly nahi dikhti wo sab yaha hoti hain.
  • Body: Ye user ka visible part hota hai, jaise headings, paragraphs, images, videos wagairah.

IV. Comments

Explanation: Comments ka use hum code me notes ya explanations likhne ke liye karte hain. Jo browser ignore kar deta hai. Ye dusre developers ko samjhne me madad karta hai.
