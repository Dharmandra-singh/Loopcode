9. HTML Forms - Full Guide

I. Form Structure

<form> tag HTML mein ek form banane ke liye use hoti hai. Form ke andar input, label, button jaise elements hote hain jo user se data collect karte hain.

  • action: Ye batata hai ki form data kis page ya URL par bhejna hai.
  • method: Ye define karta hai data bhejne ka method - GET ya POST.
  • label: User ko input ka naam dikhata hai, aur accessibility ke liye important hota hai.

II. Input Field Types

Input tag ke type attribute se hum batate hain kis tarah ka data input hoga:

  • text: Normal text
  • email: Email input with validation
  • password: Hidden input (for passwords)
  • radio: Ek hi option select karne ke liye
  • checkbox: Multiple options select karne ke liye
  • date: Date picker
  • file: File upload karne ke liye

III. Validation Basics

Form mein validation ke liye HTML attributes ka use hota hai jaise:

  • required: Field empty nahi chhoda ja sakta.
  • min, max: Numbers ke liye limit set karta hai.
  • pattern: Regular Expression ke through matching.
