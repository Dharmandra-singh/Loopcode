11. Responsive Design

I. Media Queries

Explanation: Media Queries ka use screen size ke hisaab se alag CSS apply karne ke liye hota hai. Jaise mobile, tablet, aur desktop ke layout alag ho sakte hain.

II. Responsive Units (vw, vh, rem, em)

Explanation: px ke bajay responsive units jaise vw (viewport width), vh (viewport height), rem (root font size), aur em (parent font size) use karte hain taaki layout flexible ho.

III. Mobile-first Design

Explanation: Mobile-first approach me hum pehle mobile ke liye design karte hain, aur baad me media queries ke through larger screens ke liye adjust karte hain.

IV. Fluid Layouts

Explanation: Fluid layouts me fixed width ki jagah percentage use ki jati hai jisse layout screen ke hisaab se adjust ho jata hai.
