8. HTML Tables

I. Basic Table Structure

Explanation: Table banane ke liye <table> tag ka use hota hai. Row ke liye <tr>, column ke liye <td> aur heading ke liye <th> use hota hai.

II. Table Borders and Padding

Explanation: Table mein border aur spacing control karne ke liye CSS ya cellpadding use karte hain.

III. Merging Cells (colspan & rowspan)

Explanation: Kisi cell ko horizontally ya vertically merge karne ke liye colspan (columns) aur rowspan (rows) ka use hota hai.
