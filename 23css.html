23. CSS for SEO

I. HTML5 Semantic Tags

Explanation: Search engines ko samajhne mein help milti hai jab hum <header>, <main>, <article> jaise semantic tags use karte hain. Ye batate hain ki content ka kya role hai page mein.

II. Viewport Settings

Explanation: Mobile-friendly pages ranking mein better perform karte hain. <meta name="viewport"> tag se responsive design possible hota hai.

III. SEO-friendly URLs

Explanation: URLs mein readable text use karo jaise /about-us ya /contact. Ye SEO friendly hota hai aur user-friendly bhi.

IV. Structured Data & Rich Snippets

Explanation: Structured data (JSON-LD format) Google ko batata hai ki page ka content kya hai. Isse search result mein rich snippets show hote hain (stars, price, etc).
