5. HTML Lists

I. Unordered List (ul)

Explanation: Jab hume list items bina kisi numbering ke dikhani ho, tab <ul> (unordered list) ka use hota hai. Har item ke aage bullet point lagta hai.

II. Ordered List (ol)

Explanation: Jab items ke order ka importance ho, jaise ranking ya steps, tab <ol> (ordered list) ka use hota hai. Isme numbering ya alphabets hote hain.

Example: Top 3 Programming Languages:

III. Description List (dl)

Explanation: Jab kisi word ya item ka meaning likhna ho, tab <dl> (description list) ka use hota hai. Iske andar:

  • <dt> - Term (jo define karna hai)
  • <dd> - Uska explanation
