16. CSS Shapes using Clip-path

I. Clip-path Circle

Explanation: clip-path se aap element ko circular ya kisi specific shape me dikh sakte ho. Yahan hum circle() ka use kar rahe hain.

II. Ellipse Shape

Explanation: ellipse() shape se oval ya ellipse create kar sakte hain using clip-path.

III. Polygon Shape

Explanation: Polygon coordinates dekar custom triangle, star, ya any polygon shape banayi ja sakti hai.

IV. Complex Shapes (Multiple Layers)

Explanation: Complex designs create karne ke liye aap multiple layered divs aur clip-path use kar sakte ho.
