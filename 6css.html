6. Typography in CSS

I. Font-Family

Explanation:
font-family se hum decide karte hain ki text kis type ka font dikhaye. Agar ek font available na ho, to dusra fallback use hota hai.
Example: Arial, 'Times New Roman', sans-serif

II. Font-Size

Explanation:
font-size se text ki size set hoti hai. Isme px, em, rem ya % ka use hota hai.
Example: 16px, 1.2em, 120%

III. Font-Weight

Explanation:
font-weight se text bold ya light dikhaya ja sakta hai. Values: normal, bold, 100â€“900.
Example: bold, 700

IV. Line-Height

Explanation:
line-height se lines ke beech vertical spacing control hoti hai. Isse readability improve hoti hai.

V. Letter-Spacing

Explanation:
letter-spacing se letters ke beech ka horizontal space badhaya ya kam kiya ja sakta hai.

VI. Text-Align

Explanation:
text-align se text ko left, right, center ya justify align kiya jata hai.

VII. Text-Transform

Explanation:
text-transform se hum text ko uppercase, lowercase ya capitalize kar sakte hain.

VIII. Web Fonts (Google Fonts)

Explanation:
Web fonts jaise Google Fonts ko use karne ke liye <link> tag se font import karte hain.
Example: Poppins, Roboto, Montserrat, etc.
