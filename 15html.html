15. File Paths in HTML

I. Absolute vs Relative Path

File paths HTML mein batate hain ki koi file (image, CSS, JS) kis location par hai. Do prakar ke paths hote hain:

  • Absolute Path: Full URL ya complete location (e.g. https://example.com/images/photo.jpg)
  • Relative Path: Current file ke comparison mein path (e.g. images/photo.jpg)

II. Best Practices

  • Website ke andar mostly relative paths use karo
  • Folder structure clear aur consistent rakho
  • Case sensitivity ka dhyan rakho (especially on Linux servers)
  • Broken path se bachne ke liye file names sahi check karo
  • Always test relative paths locally before deployment
