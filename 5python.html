Chapter 5: Python Loops

for, while Loops & Break, Continue, Nested Loops

1. for Loop

Kya karta hai: Yeh loop ek range ya iterable (list, tuple, string, etc.) ke elements par iterate karta hai.

Rule: for loop ka use tab karo jab aapko ek range ya collection ke elements ko process karna ho.
# for loop example for i in range(1, 6): print(i)
1 2 3 4 5
# for loop with wrong range for i in range(1, 6, 2): print(i)
1 3 5

2. while Loop

Kya karta hai: Yeh loop ek condition ko check karta hai aur jab tak wo condition true hoti hai, tab tak loop run hota hai.

Rule: while loop ko use karo jab aapko kisi condition ke true hone tak repeat karna ho.
# while loop example i = 1 while i <= 5: print(i) i += 1
1 2 3 4 5
# while loop with wrong condition i = 6 while i <= 5: print(i) i += 1
(no output, loop never runs)

3. break Statement

Kya karta hai: Yeh loop ko forcefully break karne ke liye use hota hai, even if the loop condition is still true.

Rule: Break statement tab use karo jab aapko loop ko kisi specific condition par rokna ho.
# break statement example for i in range(1, 6): if i == 3: break print(i)
1 2

4. continue Statement

Kya karta hai: Yeh statement loop ke current iteration ko skip karne ke liye use hota hai, aur next iteration par jump kar jata hai.

Rule: Continue statement tab use karo jab aap kisi iteration ko skip karna chahte ho.
# continue statement example for i in range(1, 6): if i == 3: continue print(i)
1 2 4 5

5. Nested Loops

Kya karta hai: Jab ek loop ke andar doosra loop hota hai, to usse nested loop kehte hain. Yeh complex iterations ke liye use hota hai.

Rule: Nested loops ko tab use karo jab ek loop ke andar multiple iterations perform karni ho.
# Nested loop example for i in range(1, 4): for j in range(1, 4): print(f"i={i}, j={j}")
i=1, j=1 i=1, j=2 i=1, j=3 i=2, j=1 i=2, j=2 i=2, j=3 i=3, j=1 i=3, j=2 i=3, j=3
