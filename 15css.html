15. CSS Filters

I. Blur

Explanation: blur() filter image ya element ko dhundla banane ke liye use hota hai.

Note: use image link to see exact effect

II. Brightness

Explanation: brightness() filter element ki brightness ko adjust karta hai. Value 0 se 1.5 tak hoti hai.

III. Contrast

Explanation: contrast() filter element ki contrast ko badhane ya kam karne ke liye use hota hai.

IV. Grayscale

Explanation: grayscale() filter element ko black-and-white banata hai.

V. Sepia

Explanation: sepia() filter element ko purani ya vintage look dene ke liye use hota hai.

VI. Hue-rotate

Explanation: hue-rotate() filter element ke colors ko rotate (shift) karta hai.

VII. Invert

Explanation: invert() filter element ke colors ko reverse (invert) karta hai.

VIII. Drop-shadow

Explanation: drop-shadow() filter element ke neeche shadow add karta hai.
