14. HTML Entities

I. Special Characters

HTML mein kuch characters jaise <, >, & ko directly likhne se HTML confuse ho sakta hai. Isliye hum HTML Entities ka use karte hain.

Entity ka syntax hota hai: &entity_name; ya &#entity_number;

  • &lt; = <
  • &gt; = >
  • &amp; = &
  • &quot; = "
  • &copy; = Â©
  • &nbsp; = non-breaking space
