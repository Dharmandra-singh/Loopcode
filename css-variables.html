14. CSS Variables (Custom Properties)

I. Defining Variables

Explanation: CSS Variables ko -- se define kiya jata hai. Ye variables kisi selector (mostly :root) ke andar declare kiye jaate hain.

II. Using Variables in Styles

Explanation: Variable ka use var(--name) se kiya jata hai. Aap ise background, font-size, padding, kisi bhi property me use kar sakte hain.

III. Global vs Local Scope

Explanation: Agar variable :root me define hai to wo global hota hai, matlab poore document me use ho sakta hai. Agar kisi selector ke andar define hai to wo local variable hai.

IV. Theming with Variables

Explanation: CSS Variables ka use themes banane ke liye hota hai. Aap dark mode aur light mode easily toggle kar sakte hain sirf variable values badalkar.
