10. Grid Layout

I. Grid Container & Items

Explanation: Grid layout ek 2-dimensional layout system hai jisme rows aur columns dono ka control hota hai. Parent ko display: grid dene par wo container ban jata hai.

II. Template Rows & Columns

Explanation: grid-template-columns aur grid-template-rows se hum define karte hain kitni columns ya rows honi chahiye aur unka size kya hoga.

III. Item Placement & Spanning

Explanation: Grid items ko specific row ya column me rakhne ke liye hum grid-column aur grid-row use karte hain.
Spanning ka matlab hai ek item multiple rows/columns tak faila ho.

IV. Auto-placement & Grid Areas

Explanation: CSS Grid automatically items ko place karta hai agar aap manually jagah define na karo.
Grid Areas ka use karke hum naam se jagah define karke items ko uss area me daal sakte hain (like template design).
